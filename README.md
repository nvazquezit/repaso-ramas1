# GIT RAMAS O CONOCIDAS COMO BRANCHES

## CREAMOS REPOSITORIO

```sh
git init
```

## ver status de los archivos

```sh

```

## alias... o git alias 
```sh
git config --global alias.st "status --short"
git config --global alias.a "add"
git config --global alias.c "commit -m"
git config --global alias.l "log --online"    
```
